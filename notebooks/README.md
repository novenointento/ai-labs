# Notebooks

Esta carpeta contiene **Jupyter Notebooks** utilizados para experimentos exploratorios.

Los notebooks permiten probar ideas rápidamente antes de convertirlas en scripts o aplicaciones más estructuradas.

## Usos típicos

* pruebas rápidas de modelos
* experimentos con embeddings
* exploración de datasets
* prototipos de pipelines

## Buenas prácticas

Antes de subir un notebook al repositorio:

* limpiar outputs
* eliminar datos sensibles
* documentar el objetivo del experimento

## Estructura posible

```
notebooks
├─ README.md
├─ model_tests.ipynb
├─ rag_experiments.ipynb
└─ embeddings_tests.ipynb
```
