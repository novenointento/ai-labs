# RAG Experiments

Esta carpeta contiene experimentos relacionados con **Retrieval Augmented Generation (RAG)**.

RAG es una técnica que combina modelos de lenguaje con recuperación de información desde documentos externos.

El objetivo es explorar diferentes arquitecturas y herramientas para construir sistemas RAG eficientes.

## Temas explorados

* indexación de documentos
* chunking de texto
* embeddings
* bases de datos vectoriales
* recuperación de contexto
* pipelines RAG completos

## Posibles tecnologías

* Chroma
* FAISS
* Qdrant
* Weaviate
* LangChain
* LlamaIndex

## Contenido esperado

Ejemplos de contenido:

* scripts de indexación
* pipelines RAG
* pruebas de recuperación
* experimentos de chunking
* comparativas entre bases vectoriales
