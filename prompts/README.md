# Prompts

Colección de prompts reutilizables para diferentes tareas.

El objetivo es mantener una biblioteca organizada de prompts útiles que puedan reutilizarse o evolucionar con el tiempo.

## Tipos de prompts

Ejemplos de categorías:

* prompts de programación
* prompts de análisis de texto
* prompts de debugging
* prompts de generación de documentación
* prompts experimentales

## Estructura sugerida

```
prompts
├─ README.md
├─ coding/
├─ debugging/
├─ analysis/
└─ system_prompts/
```

## Buenas prácticas

Para cada prompt se recomienda documentar:

* objetivo
* modelo utilizado
* resultado esperado
* posibles variantes
