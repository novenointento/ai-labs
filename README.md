AI Lab

Laboratorio personal de experimentación con Inteligencia Artificial.

Este repositorio recoge pruebas, herramientas, ideas y experimentos relacionados con el uso práctico de IA en entornos reales de desarrollo y arquitectura de sistemas.

El objetivo no es construir un único proyecto, sino mantener un entorno organizado donde explorar tecnologías, probar modelos, comparar enfoques y documentar aprendizajes.

Estructura del laboratorio
ai-lab
├─ models
├─ rag
├─ agents
├─ datasets
├─ prompts
├─ docker
├─ notebooks
└─ experiments
models

Contiene modelos locales, configuraciones, scripts de descarga y pruebas relacionadas con LLMs.

Ejemplos:

configuraciones de Ollama

modelos cuantizados

pruebas de rendimiento

comparativas entre modelos

rag

Implementaciones y experimentos de Retrieval Augmented Generation.

Incluye:

pipelines RAG

pruebas con bases vectoriales

indexación de documentos

experimentos de recuperación de contexto

agents

Implementación de agentes autónomos o workflows asistidos por IA.

Ejemplos:

agentes para análisis de logs

automatización de tareas

integración con APIs o herramientas externas

datasets

Datos utilizados en experimentos.

Puede incluir:

documentos para RAG

datasets de prueba

datos sintéticos

No se incluirán datasets grandes en el repositorio cuando no sea necesario.

prompts

Colección de prompts útiles para diferentes tareas.

Incluye:

prompts de programación

prompts de análisis

prompts para generación de contenido

prompts experimentales

Los prompts se guardan versionados para poder reutilizarlos y mejorarlos.

docker

Configuraciones reproducibles para entornos de pruebas.

Ejemplos:

docker-compose para LLMs locales

entornos con bases vectoriales

stacks de experimentación

El objetivo es poder levantar un laboratorio completo rápidamente.

notebooks

Experimentos exploratorios realizados con Jupyter Notebooks.

Se utilizan para:

análisis rápidos

pruebas de librerías

prototipos de pipelines de IA

experiments

Zona libre para prototipos o pruebas rápidas.

Aquí pueden aparecer:

scripts experimentales

comparativas entre técnicas

pruebas de rendimiento

ideas en fase inicial
