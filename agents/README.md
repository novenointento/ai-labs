# Agents

Esta carpeta contiene experimentos relacionados con **agentes de inteligencia artificial**.

Un agente es un sistema que utiliza un modelo de lenguaje junto con herramientas externas para ejecutar tareas complejas.

## Objetivos

Explorar cómo los LLM pueden interactuar con:

* APIs
* sistemas locales
* herramientas externas
* bases de datos
* sistemas de archivos

## Tipos de agentes posibles

Ejemplos de agentes que pueden desarrollarse aquí:

* agentes de análisis de logs
* agentes de automatización
* agentes de búsqueda de información
* asistentes técnicos
* agentes para workflows de desarrollo

## Tecnologías potenciales

* LangChain Agents
* OpenAI / LLM function calling
* CrewAI
* AutoGen
* agentes personalizados
