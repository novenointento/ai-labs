# Experiments

Zona de experimentación rápida.

Aquí se almacenan prototipos, pruebas o ideas en fase temprana que todavía no tienen una estructura clara.

## Propósito

Permitir experimentar libremente sin afectar al resto del laboratorio.

Con el tiempo, los experimentos más maduros pueden moverse a otras carpetas como:

* `rag`
* `agents`
* `models`

## Tipos de contenido

Ejemplos:

* scripts experimentales
* pruebas de librerías
* comparativas entre técnicas
* benchmarks rápidos
* prototipos

## Estructura posible

```
experiments
├─ README.md
├─ embedding_tests/
├─ inference_benchmarks/
└─ tool_prototypes/
```

Esta carpeta funciona como un **sandbox de investigación**.
